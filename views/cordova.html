Cordova

  1. Install Cordova via npm
  2. Create app by using Cordova cordova create appName appUrl appID cordova platform add android --save cordova build android cordova prepare // build the static file
  3. Install Java SDK 8 (instead of SDK 9, which will cause error)
    1. config JAVA_HOME
    2. config Path
    3. config CLASSPATH
  4. Install Android Studio use custom: include AVD
  5. Install Android SDK
