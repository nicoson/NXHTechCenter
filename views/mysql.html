MySQL

  • Installation under Ubuntu

    1. installation

      $ sudo apt-get install mysql-server

    2. test connection

      $ sudo netstat -tap | grep mysql

  • Commands Tips

    1. start/stop mysql service

      $ /etc/init.d/mysql start；(stop, restart)

    2. login mysql

      $mysql -uusername -ppassword

      or

      $mysql -uusername -p
      note: no space between u and username.

    3. show all database

      > show databases;

    4. choose to enter a database

      > use database_name;

    5. show all the table under current database

      > show tables;

    6. show the table from other database

      > show tables from db_name;

    7. show the schema of the table

      > show columns from table_name;

    8. create database

      > create database database_name;

    9. set the character set when creating database

      > create database database_name character set = utf8;

    10. change the character set after create database

      > alter database database_name character set = utf8;

    11. show character set (utf8)

      > show create database database_name;

    12. delete database

      > drop database database_name;

    13. create table

      > create table table_name(
column_name data_type primary key,
column_name data_type not null / null,
...
);

    14. keys

      primary key
unique key
default
null / not null

    15. delete a table

      > drop table table_name;

    16. insert data into table by the whole piece

      > insert into table_name values(value1, value2),(value1, value2),...,(value1, value2);

    17. insert for specific column

      > insert into table_name(col1,col2) values(value1,value2);

    18. insert ignore the duplicated rows

      > insert ignore into table_name(col1,col2) values(value1,value2);

    19. update data in table

      > update table_name set col1='value1',col2='value2' where where_definition;

    20. query language

      > select * from table_name where.......

    21. delete data from table

      > delete from table_name where...

    22. add column into created table

      > alert table table_name add column field1 date ,add column field2 time...

    23. delete column from created table

      > alert table table_name drop field1;

    24. limit

      > select * from table_name limit 3,4；
      show the following 4 results start from the 3rd result, can be used for pagination

    25. order

      > select * from table_name order by field1,orderby field2;

    26. exit mysql

      > exit;

      or

      > quit;

      or

      > q;

    27. backup database

      > $ mysqldump -uxxxx -pxxxx database_name > \home\xxx\database_name.sql

    28. recover database

      mysql> source /dir/database_name.sql
