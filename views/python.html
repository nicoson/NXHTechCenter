Install Libs for Python

  1. pip

    almost all of the libs can be installed by apt-get, but by using pip, you can get the most latest version of libs.

    $ sudo apt-get install python-pip
    $ pip install -U pip //upgrade pip
    reboot OS

  2. Ubuntu dependencies

    One command install for all dependencies:
    $ sudo apt-get install build-essential python-dev gfortran swig libatlas-dev liblapack-dev libfreetype6 libfreetype6-dev graphviz libgraphviz-dev pandoc libxml2-dev libxslt-dev zlib1g-dev

    For details:
    1. General development/build: for 3rd party libs
      $ sudo apt-get install build-essential python-dev
    2. Compilers/code integration:
      $ sudo apt-get install gfortran
      $ sudo apt-get install swig
    3. Numerical/algebra packages:
      $ sudo apt-get install libatlas-dev
      $ sudo apt-get install liblapack-dev
    4. Fonts (for matplotlib)
      $ sudo apt-get install libfreetype6 libfreetype6-dev
      $ sudo apt-get install libxft-dev
    5. Graphviz for pygraphviz, networkx, etc.
      $ sudo apt-get install graphviz libgraphviz-dev
    6. IPython require pandoc for document conversions, printing, etc.
      $ sudo apt-get install pandoc
    7. Tinkerer dependencies
      $ sudo apt-get install libxml2-dev libxslt-dev zlib1g-dev

  3. NumPy

    numpy is one of the fundamental numerical packages in Python.

    $ pip install --user numpy
    $ pip show numpy

  4. SciPy

    scipy has many useful mathematical utilities.

    $ pip install --user scipy

  5. SymPy

    sympy is a computer algebra system for Python.

    $ pip install --user sympy

  6. Matplotlib

    matplotlib is one of the main plotting packages for Python and many other packages use the utilities.

    $ pip install --user matplotlib

  7. IPython

    ipython is a major tool for sharing python projects in an interactive format.

    $ pip install --user ipython[notebook]

    launch ipython

    $ ipython notebook

  8. Spyder GUI

    $ sudo apt-get install spyder

  9. MySQLdb

    $ sudo apt-get install python-mysqldb
