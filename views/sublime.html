install & config Sublime in Ubuntu

  1. download "tar.bz2" package
  2. unzip file $ tar -xvf Sublime\ Text\ 2.0.2\ x64.tar.bz2
  3. move the file directory to /usr/local/lib $ sudo mv Sublime\ Text\ 2 /usr/local/lib
  4. set shortcut link $ sudo ln -s /usr/local/lib/Sublime\ Text\ 2/sublime_text /usr/bin/sublime
